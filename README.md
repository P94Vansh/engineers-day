# 🚀 Engineers' Day Event Registration Platform

A **full-stack event registration platform** for **Engineers' Day at Uttaranchal Institute of Technology**, built with Next.js, Tailwind CSS, Aceternity UI, and MongoDB. The platform allows students to register for events dynamically, manages team sizes and charges, and includes email verification and a protected dashboard.

## 🌟 Features
- **Dynamic Registration:** Adapts to the number of participants in each event and calculates charges accordingly.
- **Email Verification:** Sends confirmation emails using Nodemailer to ensure authentic registrations.
- **Protected Dashboard:** Uses middleware-based route protection for admin and user dashboards.
- **Manual + Dynamic Handling:** Flexible registration system for different event types.
- **Responsive UI:** Built with Tailwind CSS and Aceternity UI for smooth user experience.

---

## 🛠 Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Aceternity UI
- **Backend:** Node.js, Express (via Next.js API routes)
- **Database:** MongoDB
- **Email Service:** Nodemailer for verification emails
- **Authentication & Security:**  middleware
